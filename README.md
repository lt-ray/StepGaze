# StepGaze：視線入力における誤選択を抑制する順次視線通過型対象選択手法

視線入力インタフェースにおけるミダス・タッチ問題（ユーザーが「見ているだけ」のつもりでも意図せず選択が確定してしまう問題）の解決を目的とした、新しい対象選択手法の提案・実装です。

複数の対象を順次視線で通過させることで初めて選択を確定するアルゴリズムを設計しました。被験者実験により、既存の滞留時間方式と比較して誤選択率の有意な低減と選択完了時間の短縮を実証しています。
